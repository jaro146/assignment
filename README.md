## Project overview

Main modules

1. **app** - base parts of app like activity, application, appModule of dependency injection
2. **core** - module of base parts, UI theming, network base classes
3. **feature** - modules of features
4. **navigation** - modules of navigation (define entries and navigation between them)
5. **shared** - modules of shared elements

---


Used

* Koin for dependency injection
* Retrofit for network layer
* Room for database
* ViewModels, StateFlows, Coroutines, Jetpack Compose, Navigation3
* Mockk, Turbine for testing
